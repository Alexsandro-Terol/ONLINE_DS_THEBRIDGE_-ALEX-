\# 🚢 BATALLA NAVAL - Hundir la Flota



\## Descripción

Juego de Hundir la Flota desarrollado en Python.





\## Descripción



Este proyecto es una implementación del clásico juego Batalla Naval

desarrollado en Python para ejecutarse en la terminal.



\## Cómo funciona el juego



1\. Se crean dos tableros de 12x12.

2\. Se colocan los barcos de forma aleatoria.

3\. El jugador dispara introduciendo coordenadas.

4\. Luego dispara la máquina.

5\. Gana quien destruya primero todos los barcos del rival.





\## Flota de barcos



Cada jugador tiene 5 barcos:



\- 2 barcos de tamaño 2

\- 2 barcos de tamaño 3

\- 1 barco de tamaño 4



\## Símbolos del tablero



O = barco

X = impacto

A = agua

\_ = agua





\## Cómo ejecutar el juego



1\. Instalar numpy



pip install numpy



2\. Ejecutar el juego



python main.py
